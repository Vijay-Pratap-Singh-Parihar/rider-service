## Rider Service

#### To build and start the application

```
docker-compose up --build
```

#### Get all the riders

```
GET /v1/riders
```

#### Get specific rider

```
GET /v1/riders/{id}
```

#### Create rider profile

```
POST /v1/riders
```

#### Update rider profile

```
PATCH /v1/riders
```

### Delete rider profile

```
DELETE /v1/riders/{id}
```
